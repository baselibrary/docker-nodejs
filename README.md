## ThoughtWorks Docker Image: nodejs

[![](http://dockeri.co/image/baselibrary/nodejs)](https://registry.hub.docker.com/u/baselibrary/nodejs/)

### Base Docker Image

* `latest`: nodejs 6.0
* `4`     : nodejs 4.4
* `5`     : nodejs 5.11
* `6`     : nodejs 6.0

### Installation

    docker pull baselibrary/nodejs

### Usage

    docker run -it --rm baselibrary/nodejs
